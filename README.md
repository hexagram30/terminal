# hexagram30/terminal

*Telnet, SSL Telnet, SSH, and secure REPLs for use by hexagram30 projects*

[![][logo]][logo-large]


## Usage

Stand-alone:
```
$ lein start-telnet
```

For use as part of a component-based system, see
[hxgm30.mush.components.terminal][comp-term].
